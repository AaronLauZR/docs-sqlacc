Database upgrade to version 224. InvoiceNow factoring company support, MyInvois enhancements, and various bug fixes. (*897*)

<!-- truncate -->

## Database

- Database upgraded to version **224**

## Updates

- **General**:
  - Increased DocNo, Ref, and DocRef fields length to 50 characters.
  - Added new asset icon to Maintain Account (#7921).

- **MyInvois (E-Invoice MY)**:
  - Added Consolidate E-Invoice Request UI.
  - Added indicator to show differences between document and E-Invoice values in Sales/Purchase documents.

- **InvoiceNow (Peppol SG)**:
  - Added support for factoring company in Peppol submission.
  - Added factoring company lookup for payee vendor ID.
  - Added VendorID field to Sales/Purchase tables and data maps.
  - Revised Cash Flow Dashboard and Dashboard.

- **AI Easy Scan (OCR)**:
  - Improved CompanyName field to support UTF-8 characters (including Chinese characters).

## Bug Fixes

- **AI Easy Scan (OCR)**:
  - Fixed Dynamic SQL Error when DocType prompt contains company name or item description with only Chinese characters (#7924).

- **Stock**:
  - Fixed 64-bit crash when opening Maintain Stock Item with BOM in split form (#7923).
